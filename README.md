# 粒子动画系统 🎨

一个基于 HTML5 Canvas 的交互式粒子动画系统，支持多种动态模式和配色方案。

## 效果预览

直接在浏览器中打开 `particle-animation.html` 即可体验。

## 动画模式

点击底部按钮或按键盘数字键切换模式：

| 模式 | 键位 | 说明 |
|------|------|------|
| 🌊 **流动** | `1` | 粒子沿柏林噪声风格的流动场运动，鼠标排斥粒子 |
| ⭐ **星座** | `2` | 粒子缓慢漂移，鼠标吸引粒子，粒子间绘制连线 |
| 🌀 **漩涡** | `3` | 粒子围绕中心旋转，鼠标按下时以鼠标位置为漩涡中心 |
| 🪲 **萤火** | `4` | 粒子随机游走，间歇脉冲发光，模拟萤火虫效果 |
| 🌈 **彩虹** | `5` | 粒子做波浪形运动，颜色随位置和时间连续变化 |

## 配色方案

点击「切换配色」按钮或按 `C` 键循环切换：

| 方案 | 色调 |
|------|------|
| 🌌 **极光** | 青蓝紫色系 |
| 🌅 **落日** | 橙黄色系 |
| 🌿 **森林** | 绿色系 |
| 💜 **霓虹** | 粉紫色系 |

## 交互方式

- **鼠标移动** — 粒子受鼠标位置影响（排斥/吸引/跟随，取决于模式）
- **鼠标点击** — 漩涡模式下按住鼠标可改变漩涡中心位置
- **触屏支持** — 支持移动端触摸操作

## 技术细节

- 纯前端实现，**零依赖**，单 HTML 文件
- 使用 Canvas 2D API 进行渲染
- 粒子数量根据视口大小自适应（60–300 粒），公式：`W × H / 8000`
- 每个粒子包含发光光晕和运动轨迹尾迹效果
- 颜色使用 `hsla()` 色彩模型，方便色相循环和亮度调节
